# Suite Capacity Platform

The definitive vertically integrated Short-Term Rental (STR) operating platform.

## Overview

The Suite Capacity Platform is a data-driven system built to optimize short-term rental operations. The core feature is the Revenue Intelligence Wizard, a multi-step tool that captures property data, analyzes market trends, and presents actionable revenue projections in a command-center-style dashboard.

## Key Features

- **Revenue Intelligence Wizard**: Multi-step lead generation and analysis flow.
- **Dynamic Market Data**: Integration with pricing and analytics services (AirDNA, PriceLabs, Replicate AI).
- **Command Center Dashboard**: Premium UI for property performance review.
- **Role-Based Access Control**: Secure access for different stakeholder levels.

## Tech Stack

- TypeScript & Next.js
- Tailwind CSS & Framer Motion
- Supabase (Database, Auth, Storage)

## Getting Started

1. Set up your environment variables in `.env.local` based on `.env.example`.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
